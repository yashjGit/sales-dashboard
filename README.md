# 📊 Business Sales Intelligence Dashboard

An interactive sales analytics dashboard built with Python & Streamlit.

## 🔍 Features
- KPI cards: Total Sales, Profit, Orders, Avg Margin
- Monthly sales trend line chart
- Profit breakdown by Category & Sub-Category
- Top 10 states by revenue
- Customer segment analysis
- Discount vs Profit scatter analysis
- Sidebar filters: Region, Category, Date Range

## 🛠️ Tech Stack
Python | Pandas | Plotly | Streamlit | MySQL-ready
